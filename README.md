# Trillionipay – Centralized Wallet for Assets  
### Major Project (MP-II) | AY 2025-26  

## 📌 Project Title  
**Trillionipay Centralized Wallet for Assets**

## 📖 Overview  
Trillionipay is a centralized wallet system designed to securely manage digital assets through a scalable microservices-based architecture.  

This repository is dedicated to maintaining **weekly project logs**, documenting progress, technical decisions, architecture understanding, module development, and integration work throughout the Major Project duration.

The purpose of this repository is to:
- Maintain structured weekly progress records
- Track learning and implementation milestones
- Document system design understanding
- Reflect individual contribution and development growth

---

## 🏗️ Project Architecture Overview  

The system follows a **Microservices Architecture** pattern with independent services communicating via APIs and event-driven mechanisms.

### Core Modules:
- 👤 **User Service**
  - User registration & onboarding
  - Authentication & authorization (JWT-based)
  - Role-based access control
  - Profile management

- 💼 **Wallet Service**
  - Asset management
  - Wallet creation & tracking
  - Transaction handling
  - Balance management

- 🔔 **Notification Service**
  - Email/SMS notifications
  - OTP verification
  - Event-triggered alerts
  - Asynchronous processing
